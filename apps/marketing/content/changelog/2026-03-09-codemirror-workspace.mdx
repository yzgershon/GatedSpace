## New Workspace Modal Revamp <PRBadge url="https://github.com/superset-sh/superset/pull/2153" />

The workspace creation modal has been redesigned with improved clarity and new worktree import settings. Branches now load instantly, and the modal preserves your settings between opens.

![New workspace modal pull requests tab](/changelog/ws-2.png)

- Worktree import settings for cloning directly into a workspace
- Branches appear immediately without a loading delay
- Modal state persists between opens

## CodeMirror Editor <PRBadge url="https://github.com/superset-sh/superset/pull/2140" />

We replaced Monaco with CodeMirror for the built-in diff and file editor. CodeMirror is ~97% smaller (150KB vs 5MB), loads significantly faster, and uses far less memory. You'll feel the difference.

![CodeMirror editor](/changelog/codemirror.png)

## Material Icon Theme <PRBadge url="https://github.com/superset-sh/superset/pull/2148" />

File icons in the sidebar and file tree now use the Material Icon Theme, bringing rich per-filetype icons familiar from VS Code. Hidden files are also always visible in the file sidebar.

![Material Icon Theme](/changelog/material-icons.png)

## Cross-Workspace Search <PRBadge url="https://github.com/superset-sh/superset/pull/2193" />

The search bar now supports searching across all open workspaces, not just the active one. Trigger it from the new search bar shortcut to find files and symbols across your entire project.

![Cross-workspace search](/changelog/search-bar.png)

## PR Check Status in Changes Sidebar <PRBadge url="https://github.com/superset-sh/superset/pull/2207" />

CI check results now appear inline in the Changes sidebar alongside your PR. See pass/fail status at a glance without switching to a browser tab.

![PR check status in sidebar](/changelog/review-checks.png)

## Improvements

- **Workspace sections drag & drop** - Organize workspaces into collapsible sections with full drag-and-drop reordering <PRBadge url="https://github.com/superset-sh/superset/pull/2067" />
- **Reorderable changes sidebar sections** - Drag the Commits, Files, and other sections in the changes sidebar to your preferred order <PRBadge url="https://github.com/superset-sh/superset/pull/2105" />
- **Diff pane selectable** - Text in the diff pane is now selectable, with a button to jump to the raw file edit <PRBadge url="https://github.com/superset-sh/superset/pull/2227" />
- **Auto-run toggle for agent tasks** - A new toggle lets you control whether agent task launches run automatically or wait for confirmation <PRBadge url="https://github.com/superset-sh/superset/pull/2271" />
- **Open Preview in workspace hover card** - The workspace hover card now includes an Open Preview button for quick access <PRBadge url="https://github.com/superset-sh/superset/pull/2270" />
- **Mark as Unread** - Terminal and tab context menus now include a "Mark as Unread" option <PRBadge url="https://github.com/superset-sh/superset/pull/2276" />
- **Terminal file links open in file viewer** - Clicking file paths printed in the terminal now opens them in the built-in file viewer by default <PRBadge url="https://github.com/superset-sh/superset/pull/2266" />
- **Requested reviewers in workspace hover card** - See who has been asked to review your PR directly from the workspace card <PRBadge url="https://github.com/superset-sh/superset/pull/2136" />
- **Hover actions for Mastra messages** - User messages in Mastra chat now show quick action buttons on hover <PRBadge url="https://github.com/superset-sh/superset/pull/2117" />
- **Browser webview right-click menu** - Right-clicking links in the browser pane now shows a context menu with open/copy options <PRBadge url="https://github.com/superset-sh/superset/pull/2208" />
- **Intel (x64) macOS support** - Desktop app builds are now available for Intel-based Macs <PRBadge url="https://github.com/superset-sh/superset/pull/2256" />
- **Recent invoices on billing page** - Your last few invoices are now listed directly on the billing settings page <PRBadge url="https://github.com/superset-sh/superset/pull/2257" />
- **MCP create workspace from existing branch** - The MCP server now supports creating a new workspace from an existing workspace's branch <PRBadge url="https://github.com/superset-sh/superset/pull/2262" />
